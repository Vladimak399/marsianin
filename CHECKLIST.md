# release checklist

Чеклист перед merge в `main` и production deploy.

## Автоматические проверки

```bash
npm run typecheck
npm run lint
npm run build
```

## Главная страница

- Открывается `/` без ошибок.
- На desktop нет круглых/орбитальных линий в карте, используются прямые координатные направляющие.
- На mobile точки `о12`, `к10`, `п7` отображаются и нажимаются.
- Кнопка `карта` возвращает из open-state на карту.

## Геолокация

### Если геолокация разрешена

- Блок показывает `ваши координаты`.
- Координаты появляются с анимацией.
- Ближайшая точка подсвечивается.
- Для ближайшей точки показывается расстояние.

### Если геолокация запрещена

- Сайт не подсвечивает ближайшую точку по demo-координатам.
- Интерфейс показывает сообщение, что геолокация не получена.
- Пользователь может выбрать точку вручную.

## Выбор точки

Проверить все точки:

- `о12`
- `к10`
- `п7`

Для каждой точки:

- Клик открывает open-state.
- Отображаются код точки, адрес и режим работы.
- Блок действий содержит:
  - открыть меню;
  - построить маршрут;
  - доставка;
  - звонок;
  - яндекс карты;
  - 2гис;
  - оставить отзыв.

## Критичный сценарий меню

Для каждой точки проверить:

- `/` → выбрать точку → `открыть меню`.
- Должен открыться правильный маршрут:
  - `/menu/o12`
  - `/menu/k10`
  - `/menu/p7`
- После клика не должно быть визуального возврата на hero.
- Hero не должен оставаться поверх menu page.

Важно: переход в меню работает через `window.location.assign(href)`. Не менять на `router.push()` без отдельной проверки.

## Страница меню

- `/menu/o12` открывается.
- `/menu/k10` открывается.
- `/menu/p7` открывается.
- `/menu/unknown` показывает 404.
- Карточки меню не показывают `undefined ₽`.
- Недоступная позиция показывает корректное состояние.
- Категории меню работают.

## Админка

- `/admin` открывается.
- Вход работает через production env:
  - `ADMIN_LOGIN`
  - `ADMIN_PASSWORD`
  - `ADMIN_SESSION_SECRET`
- `Скачать JSON` скачивает актуальное меню.
- `Загрузить JSON` показывает сводку по категориям и позициям.
- После `Загрузить JSON` → `Сохранить` → refresh изменения остаются.
- Кнопки `Выше` / `Ниже` меняют порядок блюд.
- Редактируемый путь картинки принимает значения вида `/images/menu/dish-name.webp`.

## Внешние действия

Для каждой точки проверить ссылки:

- `построить маршрут` открывает карту.
- `яндекс карты` открывает Яндекс Карты.
- `2гис` открывает 2ГИС.
- `доставка` открывает Яндекс Еду.
- `звонок` использует `tel:`.
- `оставить отзыв` ведёт на страницу отзывов.

## Vercel

- Production deploy запускается только после merge в `main`.
- Preview deploys для веток могут пропускаться через `ignoreCommand`.
- Если Vercel показывает `build-rate-limit`, проверить локальную сборку перед выводами о коде.

## После merge

- Проверить production URL.
- Повторить критичный сценарий меню.
- Проверить админский сценарий JSON.
- Проверить, что GitHub checks зелёные или failure относится только к Vercel build quota.
