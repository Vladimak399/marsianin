# Марсианин — landing для кофейни

Production-ready проект на **Next.js (App Router) + TypeScript + TailwindCSS**.

## Стек

- Next.js 14
- React 18
- TypeScript
- TailwindCSS

## Структура

```bash
/app
  layout.tsx
  page.tsx
/components
  /ui
    PrimaryButton.tsx
  /menu
    MenuSection.tsx
    MenuCard.tsx
    NutritionTable.tsx
/data
  menu.ts
/styles
  globals.css
/public
  favicon.svg
  /images
```

## Запуск

```bash
npm install
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000).

## Дополнительно

- SEO metadata добавлены в `app/layout.tsx`
- favicon добавлен в `public/favicon.svg`
- изображения оптимизируются через `next/image` и современные форматы (`avif`, `webp`) в `next.config.mjs`

## Деплой на Vercel

Если после подключения GitHub-репозитория Vercel показывает `404: NOT_FOUND`, проверьте:

- Root Directory указывает на корень проекта (где лежит `package.json`);
- Framework Preset = **Next.js**;
- в репозитории есть `vercel.json` с `framework: "nextjs"`.

В этом проекте `vercel.json` уже добавлен, чтобы Vercel не определял пресет как `Other`.

## Summary: координатный вход по QR

- Интро должно быть коротким (1.4–1.8 c), не как «космический аттракцион», а как премиальная навигация.
- Сценарий: сигнал → стилизованная сфера/карта → зум в Калининград → фиксация узла (O12/K10/P7) → мягкий переход в меню.
- Меню и данные точки загружаются параллельно интро, чтобы анимация маскировала загрузку, а не блокировала UX.
- Полная версия анимации — только при входе по QR; при переключении точек внутри сайта — сокращённая (400–700 мс).
- В интерфейсе после интро сохраняются координатные элементы: код точки, адрес, координаты, тонкая сетка/crosshair.
- Для производительности: SVG/CSS/Motion и лёгкие canvas-эффекты вместо тяжёлого 3D/WebGL и больших видео.
- Для каждой точки заранее хранить: код, название, адрес, lat/lng, подпись для интро; маршрут вида `/menu/o12` должен сразу знать нужный сценарий.
- Стиль: светлый фон, тонкая сетка, монохромная графика, аккуратный оранжевый акцент, строгая типографика.
- Ограничения: не дольше 2 секунд, без резких вспышек, без полной повторной анимации на каждый клик.
